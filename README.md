jquery-slideup-基于jquery的文字自动向上滚动插件
===================================
1、引用
----------------------------------- 
在head中引用jquery-slideup.min.css和jquery-slideup.min.css

2、使用
----------------------------------- 
与其它使用jquery的方法一样：$(ele).slideUp(parms)，参数说明如下：
### parms说明：<br>
    time: num ,             // 展示时间
    movetime: num      //  滚动时间
